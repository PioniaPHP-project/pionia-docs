```bash
composer create-project pionia/pionia-app pionia-shop
cd pionia-shop && php pionia serve
```

Requires **PHP 8.5+** and `pionia/pionia-core` **^3.0**. Every guide uses **Pionia Shop** — a fictional online store + wallet — so examples stay consistent from first ping to deploy. New to PHP? See [PHP basics](/documentation/getting-started/php-basics/).
